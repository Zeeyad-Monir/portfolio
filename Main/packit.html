PackIt

Role: UX/UI Designer
Tools: Figma, Miro, Adobe Creative Suite
Date: Sept 2024 - Dec 2024
Figma Slide Deck

Overview

PackIt is an application designed to bridge the gap between large institutions with surplus food and student volunteers willing to redistribute it. The project focused heavily on creating intuitive user journeys for two distinct user groups: the institutions (donors) and the volunteers (distributors).

Design Process

This was a Figma-led prototype developed through a rigorous human-centered design process. We utilized Miro for brainstorming and Adobe Creative Suite for asset generation.

  • User Research: Conducted interviews to understand the pain points of food redistribution logistics.
  • Iterative Feedback: Moved from low-fidelity wireframes to high-fidelity mockups based on continuous testing cycles.
  • Data Validation: Validated primary user flows using simple data analysis derived from prototype testing sessions.

The Solution

  • Streamlined Discovery: Mapped end-to-end user journeys that simplify how volunteers find available shifts and how institutions post surplus.
  • Accessible UI: Designed with accessibility in mind, ensuring the interface is usable for a diverse range of users in high-paced environments.
