About SznLuv

SznLuv is a seasonal date-planning app designed for Vancouver couples. It curates unique date spots for every season, ensuring your plans match the local vibe and weather.

Core Features:

  • Seasonal Date Listings: Tailored recommendations for Fall, Winter, Spring, and Summer.
  • Interactive Maps: Google Maps integration for easy navigation.
  • Weather Forecasts: 7-day forecasts to help you plan outings effectively.
  • Spin-The-Wheel Randomizer: Instantly generate a fun date idea.
  • User Authentication & Favorites: Secure sign-up (Face ID on iOS) with a Favorites system.

APIs & Technologies: Google Maps API (planned), OpenWeather API, Firebase Authentication, Firestore, and React Native with Expo.

For more details, visit the live app page or check out the GitHub repository.
