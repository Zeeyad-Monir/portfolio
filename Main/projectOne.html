About SznLuv

SznLuv is a comprehensive seasonal date planning app designed for Vancouver couples. It has curated a list of unique date spots tailored to each season, whilst showing local weather,directions, and map, ensuring every outing is perfectly timed and styled.

Key Features:

  • Seasonal Date Listings: Curated recommendations for Fall, Winter, Spring, and Summer.
  • User Authentication & Profile Management: Secure sign-up with Firestore, customizable profiles, and easy logout.
  • Real-Time Navigation: Native GPS tracking with Apple Maps enhanced by Google Maps API for directions.
  • Weather Integration: A 5-day forecast with temperature unit options to help plan accordingly.
  • Interactive Spin-The-Wheel: A fun, randomized game for selecting date spots, with previous results stored temporarily.(Gameification)
  • Personalized Favourites & Reviews: Save and categorize date spots, and benefit from community driven reviews.
  • Visual Enhancements: Engaging Lottie animations and smooth UI transitions add a dynamic touch without overwhelming the interface.

APIs & Technologies: SznLuv utilizes Google Maps API, OpenWeather API, Firebase Authentication, Firestore, and React Native with Expo to create a seamless and responsive user experience.
