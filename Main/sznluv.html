SznLuv

Role: Developer & Designer

Tools: React Native, Expo SDK, JavaScript, Figma, Miro

Date: Sept 2024 - Dec 2024

View Demo Figma Slide Deck Report GitHub
01 / Overview

SznLuv is a seasonal discovery mobile application designed specifically for Vancouver couples. The app solves the everyday question of "what should we do today?" by providing curated date ideas based on the current season and weather, allowing users to save their favourites for later.

02 / Design Process
01

Concept & Research

Identified the core problem couples face when planning dates, especially with Vancouver's seasonal weather. Researched existing solutions and found a gap in location-aware, weather-responsive recommendations.

02

Design & Prototyping

Created wireframes and high-fidelity mockups in Figma, using Miro for brainstorming sessions. Focused on a clean, intuitive interface that feels native to both iOS and Android.

03

Technical Development

Built the application using React Native and JavaScript, leveraging the Expo SDK for rapid prototyping, efficient build processes, and cross-platform compatibility.

04

API Integration

Integrated external map services for location-based discovery alongside weather APIs to suggest activities appropriate for the current forecast, delivering real-time value to users.

Reasoning

Designed around how couples actually plan. Each feature is grounded in research on decision-making, context-aware computing, and relationship psychology.

03 / The Solution

Map-Based Discovery

Location-detail flows let users visually explore date spots in their vicinity. Tap any pin to dive into specifics, see photos, and get directions instantly.

Privacy-First Onboarding

A streamlined sign-in flow that prioritizes user privacy while reducing initial friction. The design led to higher conversion rates during prototype testing.

Saves & Collections

A robust "Favorites" system allowing couples to curate their own lists of seasonal activities. Save spots you want to try and revisit them anytime.
