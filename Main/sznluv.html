SznLuv

Role: Developer & Designer
Tools: React Native, Expo SDK, JavaScript, Figma, Miro
Date: Sept 2024 - Dec 2024
Demo Figma Slide Deck Project Report GitHub

Overview

SznLuv is a seasonal discovery mobile application designed specifically for Vancouver couples. The app solves the problem of "what should we do today?" by providing curated date ideas and allowing users to save their favourites for later.

Technical Implementation

Built using React Native and JavaScript, the application leverages the Expo SDK to handle cross-platform compatibility. The core functionality relies on complex API integrations to deliver real-time value to the user.

  • Map & Weather APIs: Integrated external map services to provide location-based discovery, alongside weather data to suggest activities appropriate for the current forecast.
  • Expo SDK: Utilized Expo for rapid prototyping and efficient build processes.

Key Features

  • Map-Based Discovery: Designed and implemented location-detail flows that allow users to visually explore date spots in their vicinity, improving findability.
  • Privacy-First Onboarding: Prototyped a sign-in flow that prioritizes user privacy while reducing initial friction, leading to higher conversion rates during testing.
  • Saves & Collections: A robust "Favorites" system allowing couples to curate their own lists of seasonal activities.
