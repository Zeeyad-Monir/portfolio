About the Project

The Arduino and Java-based AC Cooling Fan project is designed to offer an intelligent cooling solution tailored to user presence and environmental conditions. By integrating temperature, distance, and water level sensors, the system dynamically adjusts the cooling intensity to ensure comfort while optimizing energy usage.

Key Highlights:

  • Three-sensor integration for comprehensive environmental monitoring
  • Automatic mode adjusts fan speed based on user proximity
  • Manual mode provides direct control over fan speeds
  • Water cooling system enhances air cooling efficiency
  • Alert system notifies users when water maintenance is required
  • Developed using Arduino for hardware control and Java for user interface

For more detailed information, check out the GitHub repository.
